# Wolfey's Teambuilding Mental Model

## Reverse-Engineered from Stream & Video Ground Truths

**Methodology note:** This document was built in two passes to avoid bias:
1. First, 15 raw patterns were extracted ONLY from `aiming-for-champions.md` (see `wolfey-raw-extraction.md`)
2. Then, those patterns were compared against the existing SKILL.md (see `skill-comparison-and-new-learnings.md`)
3. Only genuinely new learnings were added. The existing skill framework was NOT used to interpret the stream.

This document distills the teambuilding progression Wolfey (WolfeyVGC) follows, extracted from his actual decision-making across multiple teams: Mega Meganium, Mega Starmie (with Alpharad), Mega Drampa, Mega Froslass, and Hisuian Zoroark (Milwaukee Regionals).

## Cross-Cutting Principles (Observed Across All Teams)

### 1. The Star ≠ The Win Condition
The Pokémon you build around is often NOT the one that closes games.

| Team | Star (creates advantage) | Win Condition (converts to KOs) |
|------|--------------------------|----------------------------------|
| Meganium | Mega Meganium (damage + pressure) | Tauros / Kangaskhan (cleanup) |
| Starmie | Mega Starmie (Skill Swap donor) | Tyranitar / Garchomp (Huge Power nukes) |
| Drampa | Mega Drampa (Berserk stacking) | Drampa itself (once at +4/+6) |
| Froslass | Mega Froslass (star AND win con — exception where they overlap) | Orthworm enables it via Shed Tail |
| Zoroark | Zoroark (Illusion disruption) | Ursaluna Bloodmoon (Blood Moon KOs) |

### 2. Speed Philosophy Is Team-Wide
Wolfey commits to a speed tier for the whole team:
- Meganium team: Tailwind-dependent. Every Pokémon benefits from doubled speed.
- Zoroark team: Icy Wind + Tailwind. Ursaluna is trained fast (not slow/bulky) to combo with speed drops.
- Drampa team: Speed Swap gives Drampa 189 speed. Backup plan is Trick Room via Sinistea.
- Froslass team: Shed Tail buys time; Froslass is naturally fast. Garchomp + Basculegion are fast independently.

### 3. Punish Passive Play
Every team has a mechanism to punish opponents who try to Protect/scout:
- Zoroark: Whimsicott's Encore locks them into Protect
- Froslass: Nasty Plot behind Substitute (free setup if they Protect)
- Drampa: Calm Mind stacking (every passive turn = Drampa gets stronger)
- Meganium: Raw offensive pressure (Tailwind + Weather Ball + Solar Beam)
- Starmie: Skill Swap happens in one turn (no setup to scout)

### 4. Information Advantage Is a Weapon
Wolfey treats information asymmetry as a real resource:
- Zoroark's Illusion creates uncertainty about which Pokémon is on the field
- Frisk (H-Typhlosion) reveals opponent items immediately
- Protecting both Pokémon to scout opponent's moves/items before committing
- "This looks like a passive do-nothing play, but it does two things: scout what move Basculegion locks into, and scout for Delphox Protect"

### 5. Items Are a Constraint, Not an Afterthought
Especially in Champions format:
- "Why are there no items in this game?"
- "White Herb is already used. Lum Berry isn't used. Chople Berry, Dragon Fang — these are all garbage."
- "Is it bad that I think Bright Powder is correct?"
- Settles on Leftovers for Archaludon by elimination → turns out to be excellent

### 6. Emotional Discipline During Iteration
- "That's okay. It's okay. It's okay." (after a miss/bad turn)
- "I think I'm losing this... although Tauros, it's now..." (stays analytical)
- "I was way too greedy. I should have protected." (immediate self-correction)
- "I think I need to take longer [in Team Preview]" (recognizes rushing)
- Doesn't rebuild after one loss. Changes one variable and re-tests.

### 7. Challenge Conventional Wisdom
Wolfey doesn't just ignore the meta — he identifies the specific flaw in the popular approach:
- Meganium: "People are pairing it with Pelipper... but Pelipper is too slow to exploit Tailwind." → Rejects Pelipper for Aerodactyl.
- Drampa: "Almost all excitement went out the window [when it got Berserk instead of Drizzle], except for me." → Finds the Berserk healing loop nobody explored.
- Zoroark: "Seeing almost zero usage. But we think it could win this tournament." → Reframes it as a disruptor, not an attacker.
- Froslass: Builds Nasty Plot set instead of the more common Aurora Veil support set.
- The disagreement with consensus IS the creative spark for divergent builds.

### 8. Pre-Mega Ability Exploitation
Wolfey deliberately times mega evolution to exploit the base form's ability as a free resource:
- Drampa: Uses Cloud Nine before mega evolving to neutralize opponent's weather for free.
- Froslass: Delays mega evolution to time Snow Warning's 5-turn window for maximum value.
- "We're not going to mega here because we don't actually want to set the snow up yet."

## The Wolfey Decision Tree (Simplified)

```
START: "I want to build around [Pokémon X]"
  │
  ├─▶ What is X's UNIQUE mechanic? (ability, move combo, typing)
  │
  ├─▶ What is X's #1 BOTTLENECK? (speed, bulk, setup time, info control)
  │     │
  │     └─▶ Pick Partner A to solve that bottleneck specifically
  │
  ├─▶ What CONSTRAINTS does this core impose?
  │     │
  │     └─▶ List them ALL before picking more Pokémon
  │
  ├─▶ For each remaining slot:
  │     │
  │     ├─▶ Does it enable the core? (support role)
  │     ├─▶ Does it threaten the core's worst matchups? (offensive role)
  │     ├─▶ Does it respect all constraints?
  │     └─▶ Does it fit the speed philosophy?
  │
  ├─▶ Dedicate one slot to the format's #1 threat (specific set, not generic)
  │
  ├─▶ Build a Plan B for when X can't be brought
  │     │
  │     └─▶ In Champions: backup Mega Evolution
  │
  └─▶ TEST → identify pain points → change ONE thing → repeat
```

## Anti-Patterns Wolfey Avoids

1. **The Goodstuff Trap**: Surrounding a unique Pokémon with the top-5 most-used Pokémon instead of ones that solve its specific problems.

2. **Forcing the Star**: Bringing Meganium into Sun because "it's the star" when Kangaskhan is clearly better for that matchup.

3. **Theorycrafting Forever**: "Get a team to 6 Pokémon, test it, then refine. Don't chase perfection in the builder."

4. **Rebuilding After One Loss**: Wolfey changes Aegislash → Archaludon (one swap), not the whole team.

5. **Ignoring Item Scarcity**: In Champions, items are limited. Two Pokémon wanting the same item means one needs a different build.

6. **Support Bots**: Picking a Tailwind setter that ONLY sets Tailwind. The best support Pokémon also threaten the core's bad matchups offensively.

7. **Single-Target Gimmicks**: If your combo only works on one recipient, the opponent counters it in Team Preview. Always have a second target.
