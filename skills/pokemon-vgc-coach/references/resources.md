# VGC Resources Directory

## Tier 1: Essential Tools (Use These Constantly)

### Pikalytics — Usage Statistics & Metagame Data
- **URL**: https://www.pikalytics.com
- **AI Endpoints** (fetch these for live data):
  - `https://www.pikalytics.com/ai` — Hub landing page
  - `https://www.pikalytics.com/ai/pokedex` — Top 50 Pokemon in default format
  - `https://www.pikalytics.com/ai/pokedex/{format}/{pokemon}` — Individual Pokemon data
  - `https://www.pikalytics.com/ai/champions` — Pokemon Champions info
- **Format Codes**:
  - `championspreview` — Pokemon Champions (current default)
  - `gen9vgc2026regf` — VGC 2026 Regulation F
  - `gen9vgc2025regi` — VGC 2025 Regulation I
  - `gen9ou` — Smogon OverUsed (singles)
- **What it provides**: Usage rates, move distributions, item choices, ability picks, EV spreads, teammate synergies, counters/checks, Tera Type preferences
- **When to use**: ALWAYS check Pikalytics before giving metagame advice. It's the single best source of truth for what's actually being used.

### Pokemon Showdown — Battle Simulator
- **URL**: https://play.pokemonshowdown.com
- **What it provides**: Free online Pokemon battles, teambuilder, damage calculator
- **When to use**: Testing teams, practicing, laddering. Recommend this as the primary testing ground.
- **Damage Calculator**: https://calc.pokemonshowdown.com

### VGCGuide.com — Fundamentals Guide
- **URL**: https://www.vgcguide.com
- **What it provides**: Comprehensive free written guide covering all VGC fundamentals
- **Key sections**:
  - Teambuilding: https://www.vgcguide.com/teambuilding
  - Synergy: https://www.vgcguide.com/synergy
  - Cores and Modes: https://www.vgcguide.com/cores-and-modes
  - Team Preview: https://www.vgcguide.com/team-preview
  - How to Pull It All Together: https://vgcguide.com/how-to-pull-it-all-together
- **Written by**: Wolfe Glick (World Champion), Aaron Traylor, and others
- **When to use**: Recommend to users learning fundamentals. Evergreen content.

## Tournament Information

### Finding Events
- **Event Locator**: Check Pokemon.com for official event locators
- **Tournament Levels** (ascending):
  1. League Challenges (monthly local events)
  2. League Cups (quarterly local events, more points)
  3. Regionals (large multi-day events, ~1000 players)
  4. International Championships (3 per season worldwide)
  5. World Championships (invite-only, end of season in August)
- **Online Events**: Grand Challenges (GC) — ladder-style, Best of 1, Closed Teamsheet
- **Unofficial Online**: Victory Road and Limitless tournaments

### Championship Points
- Earned at sanctioned events based on placement
- Used to qualify for World Championships
- Top earners receive travel stipends for major events
- Ace Trainer Rewards for reaching point thresholds
