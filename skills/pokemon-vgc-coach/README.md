<h1 align="center">Pokemon VGC Coach</h1>

<p align="center">AI coaching skill for competitive Pokemon VGC — teambuilding, metagame analysis, damage calculations, and in-battle strategy for doubles battles.</p>

## Capabilities

- **Teambuilding** — 5-step framework from core idea to final details
- **Metagame Analysis** — Live usage stats and top sets from Pikalytics
- **Damage Calculations** — Inline calcs for EV benchmarks and KO ranges
- **Synergy Library** — 30+ documented mechanical combos
- **In-Battle Coaching** — Team preview, turn-by-turn theory, endgame calculation
- **Format-Aware** — Pokemon Champions, Scarlet/Violet regulations

## MCP Tools

| Tool | Purpose |
|------|---------|
| `calculateDamage` | Damage calculations for EV spreads and KO ranges |
| `create_pokepaste` | Export teams to PokePaste for sharing |

## Data Sources

- **Metagame data:** [Pikalytics](https://www.pikalytics.com) (live usage stats, sets, teammate data)
- **Coaching philosophy:** [WolfeyVGC](https://www.youtube.com/@WolfeyVGC) and [VGCGuide.com](https://www.vgcguide.com)
